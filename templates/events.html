Certification Path

AWS Cert Roadmap

The recommended certification path we guide our members through.

Cloud Practitioner

Entry-level. Core AWS concepts, pricing, and cloud fundamentals.

Solutions Architect – Associate

Design resilient, high-performing, and cost-efficient architectures.

Developer – Associate

Build, deploy, and debug cloud-based applications using AWS services.

SysOps – Associate

Manage and operate systems on AWS, monitoring, and automation.

Professional / Specialty

Advanced paths — Solutions Architect Pro, Security, ML, and more.
