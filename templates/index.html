Who We Are

Built by students,
powered by the cloud.

AWS Cloud Club - Flurry is an official AWS Cloud Club chapter dedicated to equipping students with real-world cloud skills. We run hands-on workshops, collaborate with other chapters, and help members earn AWS certifications that matter.

Whether you're a complete beginner or already building on AWS, there's a place for you here.

Meet the Team

Community First

A welcoming space for IT and CS students to connect, collaborate, and grow together.

Hands-On Learning

Every workshop is built around doing — not just watching. You leave with real skills.

Cert-Driven

We guide members toward AWS certifications with structured roadmaps and study sessions.

AWS Official Chapter

Recognized and supported by AWS, connected to a national network of cloud clubs.

What You'll Learn

Core AWS Topics We Cover

From fundamentals to advanced architecture — here's what our workshops focus on.

EC2 & Compute

Launch and configure virtual servers, understand instance types, auto-scaling, and load balancing.

S3 & Storage

Store and serve data at scale with S3. Learn buckets, policies, versioning, and static hosting.

Lambda & Serverless

Build event-driven functions without managing servers. Deploy with API Gateway and triggers.

RDS & Databases

Work with managed relational databases, DynamoDB for NoSQL, and best practices for data modeling.

IAM & Security

Manage users, roles, and permissions. Understand least-privilege access and security best practices.

VPC & Networking

Design secure cloud networks with VPCs, subnets, route tables, and security groups.

CloudWatch & Monitoring

Set up logs, alarms, and dashboards to observe and troubleshoot your cloud infrastructure.

CI/CD & DevOps

Automate deployments with CodePipeline, CodeBuild, and integrate with GitHub for cloud workflows.
